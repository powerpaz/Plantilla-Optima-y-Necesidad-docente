
# PO-App v4

Aplicativo web para validación normativa y revisión de Plantilla Óptima.

Incluye:

- Validación Acuerdo 2023-00086-A
- Validación Acuerdo 2024-00065-A (vigente)
- Motor de equivalencias oficiales
- Lectura Excel (hoja NOMINE, celda B2 ejemplo)

 ├── styles.css
 ├── config.js
