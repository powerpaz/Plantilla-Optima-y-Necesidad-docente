
# PO-Check v2

Implementa validación normativa automática:

Si una IE declara una figura profesional que:

- Existe en 00086 pero NO está en 00065 → observación crítica.
- Cambió de denominación → validación con equivalencias.
- Fue suprimida → inconsistencia normativa.

## Resumen Ejecutivo

| Elemento | 2023-00086-A | 2024-00065-A |
|-----------|--------------|--------------|
| Tipo | Catálogo anterior | Catálogo actualizado |
| Vigencia actual | Superado | Vigente |
| Uso para PO | No recomendado | Obligatorio |
| Impacto en distributivo | Base histórica | Base oficial actual |
